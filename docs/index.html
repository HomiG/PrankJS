Features

Modular Design

Import only the pranks you need, keeping your bundle size small.

TypeScript Support

Fully typed API for better developer experience and code completion.

Installation

  • Install with npm
    bash npm install prankjs
  • Import in your project
    javascript import { Pranks } from 'prankjs';
  • Use any prank function
    javascript // Examples: Pranks.shakeElement('.navigation'); Pranks.invertColors(3000, 0, 1); Pranks.fakeError('This is a fake error message!'); Pranks.randomFloatingObjects('https://example.com/image.png');
